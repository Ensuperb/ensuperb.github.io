Python 字符串前面加u,r,b的含义

  • u

    • 以 Unicode 格式 进行编码，一般用在中文字符串前面
    • 例子:u”我是含有中文字符组成的字符串。”

  • r

    • 去掉反斜杠的转移机制。
    • r”\n\n\n\n”

  • b

    • bytes 类型
    • response = b’Hello World!’
